# GitHub Pages 可直接上線版

## 使用方式
1. 到 GitHub 建立一個新 repository
2. 把這個資料夾中的所有檔案上傳到 repository 最外層
3. 到 **Settings → Pages**
4. Source 選 **Deploy from branch**
5. Branch 選 **main**，Folder 選 **/root**
6. 儲存後稍等 1～5 分鐘
7. 打開 GitHub Pages 網址即可

## 重要
首頁檔案必須是：
- index.html

## 修改位置
- 主要文字內容：`index.html`
- 抽牌功能：`index.html` 最底部 `<script>` 區塊
- 顏色設定：`index.html` 最上方 `:root`

## 這個版本已包含
- GitHub Pages 首頁
- 404 備用頁
- 橘色選單切換效果
- 1 / 3 / 5 張抽牌互動示範
