# GitHub Pages 還原感版本

## 這次修正
- 改回較接近原本的深紫色系
- 保留橘色被點選效果
- 補回較完整網站選項
- 首頁依然使用 index.html，適合 GitHub Pages

## 使用方式
1. 建立 GitHub repository
2. 把本資料夾所有檔案上傳到最外層
3. 到 Settings → Pages
4. Source 選 Deploy from branch
5. Branch 選 main，Folder 選 /root
6. 儲存後稍等數分鐘即可上線

## 可修改區
- 所有網站文字：index.html
- 顏色：index.html 的 :root
- 抽牌內容：index.html 最底部 script
